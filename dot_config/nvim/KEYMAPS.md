# Colemak 键位配置

基于 LazyVim 的 Colemak 键位映射，核心理念是将 QWERTY 的 `hjkl` 导航替换为 Colemak 友好的 `neia` 菱形布局。

## 导航布局

```
      u
      ↑
  n ← · → i
      ↓
      e
```

## Leader 键

**Leader 键 = `空格键`**

所有 `<Leader>` 开头的快捷键都需要先按空格。

---

## 核心导航

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `u` | 上移一行 | `k` | n, v |
| `e` | 下移一行 | `j` | n, v |
| `n` | 左移一字符 | `h` | n, v |
| `i` | 右移一字符 | `l` | n, v |
| `gu` | 软换行上移 | `gk` | n, v |
| `ge` | 软换行下移 | `gj` | n, v |

## 快速导航

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `U` | 快速上移 5 行 | `5k` | n, v |
| `E` | 快速下移 5 行 | `5j` | n, v |
| `N` | 跳转到行首 | `0` | n, v |
| `I` | 跳转到行尾 | `$` | n, v |
| `h` | 跳转到词尾 | `e` | n, v |
| `W` | 前进 5 个单词 | `5w` | n, v |
| `B` | 后退 5 个单词 | `5b` | n, v |

## 视图滚动

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|

| `Ctrl+E` | 向下滚动视图 5 行 | `5<C-e>` | n, v |

|--------|------|-----------|------|
| `k` | 进入插入模式 | `i` | n |
| `K` | 行首进入插入模式 | `I` | n |

## 插入模式快捷键

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `Ctrl+a` | 移至行末 | `<End>` | i |
| `Ctrl+u` | 删除至行末 | `<C-o>d$` | i |

## Operator-pending 模式

用于 `c`、`d`、`y` 等操作符后的移动和文本对象选择。

### 导航

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `u` | 上移 | `k` | o |
| `e` | 下移 | `j` | o |
| `n` | 左移 | `h` | o |
| `i` | 右移 | `l` | o |

### 文本对象

| 快捷键 | 功能 | 原 Vim 键 | 示例 |
|--------|------|-----------|------|
| `k` | inner 文本对象 | `i` | `ckw` = `ciw`（修改单词内部） |
| `a` | around 文本对象 | `a` | `caw`（修改整个单词） |

> 由于 `i` 被用作右移，使用 `k` 代替 `i` 作为 inner 前缀：
> - `ckw` = 修改单词内部 (原 `ciw`)
> - `dkw` = 删除单词内部 (原 `diw`)
> - `ykw` = 复制单词内部 (原 `yiw`)
> - `ck"` = 修改引号内部 (原 `ci"`)

## 可视模式快捷键

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `Y` | 复制至系统剪切板 | `"+y` | v |
| `k` | 块插入（在选区前） | `I` | x (Visual Block) |

> Visual Block 模式 (`Ctrl+v`) 下按 `k` 进入块插入模式。

## Surround 操作 (mini.surround)

对文本添加、删除、替换包围符号。

### 可视模式（选中文本后）

| 快捷键 | 功能 | 示例 |
|--------|------|------|
| `gsa"` | 添加双引号 | `text` → `"text"` |
| `gsa'` | 添加单引号 | `text` → `'text'` |
| `` gsa` `` | 添加反引号 | `text` → `` `text` `` |
| `gsa)` | 添加圆括号 | `text` → `(text)` |
| `gsa]` | 添加方括号 | `text` → `[text]` |
| `gsa}` | 添加花括号 | `text` → `{text}` |

### 普通模式

| 快捷键 | 功能 | 示例 |
|--------|------|------|
| `gsaiw"` | 给当前单词添加双引号 | `word` → `"word"` |
| `gsd"` | 删除最近的双引号 | `"text"` → `text` |
| `gsr"'` | 替换双引号为单引号 | `"text"` → `'text'` |
| `gsf"` | 查找右侧双引号 | - |
| `gsF"` | 查找左侧双引号 | - |
| `gsh"` | 高亮双引号 | - |

> 配置文件：`~/.config/nvim/lua/plugins/surround.lua`

## 窗口管理

### 窗口切换

| 快捷键 | 功能 | 原 Vim 键 |
|--------|------|-----------|
| `<Leader>u` | 切换到上方窗口 | `<C-w>k` |
| `<Leader>e` | 切换到下方窗口 | `<C-w>j` |
| `<Leader>n` | 切换到左方窗口 | `<C-w>h` |
| `<Leader>i` | 切换到右方窗口 | `<C-w>l` |

### 分屏

| 快捷键 | 功能 |
|--------|------|
| `su` | 向上分屏（新窗口在上） |
| `se` | 向下分屏（新窗口在下） |
| `sn` | 向左分屏（新窗口在左） |
| `si` | 向右分屏（新窗口在右） |
| `sv` | 两分屏垂直放置（左右排列） |
| `sh` | 两分屏水平放置（上下排列） |

> 注：`s` 键本身已禁用，仅作为分屏前缀使用。

## Tab 管理

| 快捷键 | 功能 |
|--------|------|
| `tu` | 新建 Tab |
| `tn` | 切换到上一个 Tab |
| `ti` | 切换到下一个 Tab |
| `tmn` | 将当前 Tab 左移 |
| `tmi` | 将当前 Tab 右移 |

## 搜索导航

| 快捷键 | 功能 | 原 Vim 键 |
|--------|------|-----------|
| `=` | 下一个搜索结果 | `n` |
| `-` | 上一个搜索结果 | `N` |

> 由于 `n/N` 被用于左移和行首，搜索导航改用 `=/-`。

## 显示切换

| 快捷键 | 功能 |
|--------|------|
| `<Leader>tw` | 切换软换行 (toggle wrap) |

> 软换行开启时，长行会在窗口边缘自动换行显示。配合 `gu`/`ge` 可在软换行间移动。

## 常用操作

| 快捷键 | 功能 | 原 Vim 键 |
|--------|------|-----------|
| `;` | 进入命令模式 | `:` |
| `Q` | 退出 | `:q<CR>` |
| `S` | 保存 | `:w<CR>` |

| Colemak | 功能 | 对应原 Vim | 模式 |
|---------|------|------------|------|
| `u` | 上 | `k` | n, v, o |
| `e` | 下 | `j` | n, v, o |
| `n` | 左 | `h` | n, v, o |
| `i` | 右 | `l` | n, v, o |
| `k` | 插入 / inner 文本对象 | `i` | n / o |
| `l` | 撤销 | `u` | n |
| `h` | 词尾 | `e` | n, v |
