# Colemak 键位配置

基于 LazyVim 的 Colemak 键位映射，核心理念是将 QWERTY 的 `hjkl` 导航替换为 Colemak 友好的 `neia` 菱形布局。

## 导航布局

```
      u
      ↑
  n ← · → i
      ↓
      e
```

## Leader 键

**Leader 键 = `空格键`**

所有 `<Leader>` 开头的快捷键都需要先按空格。

---

## 核心导航

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `u` | 上移一行 | `k` | n, v |
| `e` | 下移一行 | `j` | n, v |
| `n` | 左移一字符 | `h` | n, v |
| `i` | 右移一字符 | `l` | n, v |
| `gu` | 软换行上移 | `gk` | n, v |
| `ge` | 软换行下移 | `gj` | n, v |

## 快速导航

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `U` | 快速上移 5 行 | `5k` | n, v |
| `E` | 快速下移 5 行 | `5j` | n, v |
| `N` | 跳转到行首 | `0` | n, v |
| `I` | 跳转到行尾 | `$` | n, v |
| `h` | 跳转到词尾 | `e` | n, v |
| `W` | 前进 5 个单词 | `5w` | n, v |
| `B` | 后退 5 个单词 | `5b` | n, v |

## 视图滚动

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `Ctrl+U` | 向上滚动视图 5 行 | `5<C-y>` | n, v |
| `Ctrl+E` | 向下滚动视图 5 行 | `5<C-e>` | n, v |

## 模式切换

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `k` | 进入插入模式 | `i` | n |
| `K` | 行首进入插入模式 | `I` | n |
| `l` | 撤销 | `u` | n |

## 插入模式快捷键

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `Ctrl+a` | 移至行末 | `<End>` | i |
| `Ctrl+u` | 删除至行末 | `<C-o>d$` | i |

## 可视模式快捷键

| 快捷键 | 功能 | 原 Vim 键 | 模式 |
|--------|------|-----------|------|
| `Y` | 复制至系统剪切板 | `"+y` | v |

## 窗口管理

### 窗口切换

| 快捷键 | 功能 | 原 Vim 键 |
|--------|------|-----------|
| `<Leader>u` | 切换到上方窗口 | `<C-w>k` |
| `<Leader>e` | 切换到下方窗口 | `<C-w>j` |
| `<Leader>n` | 切换到左方窗口 | `<C-w>h` |
| `<Leader>i` | 切换到右方窗口 | `<C-w>l` |

### 分屏

| 快捷键 | 功能 |
|--------|------|
| `su` | 向上分屏（新窗口在上） |
| `se` | 向下分屏（新窗口在下） |
| `sn` | 向左分屏（新窗口在左） |
| `si` | 向右分屏（新窗口在右） |
| `sv` | 两分屏垂直放置（左右排列） |
| `sh` | 两分屏水平放置（上下排列） |

> 注：`s` 键本身已禁用，仅作为分屏前缀使用。

## Tab 管理

| 快捷键 | 功能 |
|--------|------|
| `tu` | 新建 Tab |
| `tn` | 切换到上一个 Tab |
| `ti` | 切换到下一个 Tab |
| `tmn` | 将当前 Tab 左移 |
| `tmi` | 将当前 Tab 右移 |

## 搜索导航

| 快捷键 | 功能 | 原 Vim 键 |
|--------|------|-----------|
| `=` | 下一个搜索结果 | `n` |
| `-` | 上一个搜索结果 | `N` |

> 由于 `n/N` 被用于左移和行首，搜索导航改用 `=/-`。

## 常用操作

| 快捷键 | 功能 | 原 Vim 键 |
|--------|------|-----------|
| `;` | 进入命令模式 | `:` |
| `Q` | 退出 | `:q<CR>` |
| `S` | 保存 | `:w<CR>` |

## 键位映射对照表

| Colemak | 功能 | 对应原 Vim |
|---------|------|------------|
| `u` | 上 | `k` |
| `e` | 下 | `j` |
| `n` | 左 | `h` |
| `i` | 右 | `l` |
| `k` | 插入 | `i` |
| `l` | 撤销 | `u` |
| `h` | 词尾 | `e` |

## 配置文件

键位配置位于：`~/.config/nvim/lua/config/keymaps.lua`
